## 최근 추가된 기능 테스트

### 1. 이미지 업로드 2개 제한 기능
- [ ] 이미지 0개 → 1개 추가 성공
- [ ] 이미지 1개 → 2개 추가 성공
- [ ] 이미지 2개 → 추가 불가 (경고 메시지)
- [ ] 이미지 삭제 후 다시 추가 가능
- [ ] 여러 파일 선택 시 슬롯 제한

### 2. 기록 상세 화면 이미지 표시 (썸네일 스트립) ⭐ 신규
- [ ] 이미지가 있는 기록 날짜 클릭
- [ ] 기록 상세 모달 표시 확인
- [ ] 이미지가 썸네일 스트립 형태로 표시 (60px x 60px)
- [ ] 입력 필드 형태의 박스 UI가 아닌지 확인
- [ ] 이미지 border-radius: 14px 확인
- [ ] 이미지 간 gap: 10px 확인
- [ ] 이미지가 없을 때는 해당 영역 렌더링 안 됨
- [ ] 이미지 hover 효과 확인

### 3. 이미지 첨부 후 저장 기능
- [ ] 이미지 1개 첨부 후 저장 성공
- [ ] 이미지 2개 첨부 후 저장 성공 (첫 번째만 저장됨)
- [ ] 기록 수정 시 이미지 첨부 후 수정 완료 성공
- [ ] 400 에러 발생하지 않음
