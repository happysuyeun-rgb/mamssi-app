# 홈 화면 "나의 정원" 카드 레이아웃 변경 사항

## 변경 요약

기존 좌우 배치 레이아웃을 세로 스택 구조로 변경하고, 게이지를 중앙에 배치했습니다.

## 변경된 파일

### 1. `src/components/home/FlowerBadge.tsx`
- JSX 구조를 세로 스택으로 재구성
- 게이지를 중앙 정렬로 배치
- "현재 성장 단계" 섹션 추가 (연한 민트 배경)
- 씨앗 이름 섹션 중앙 정렬
- 오늘의 정원 소식 하단 배치

### 2. `src/styles/home.css`
- 세로 스택 레이아웃 스타일 추가
- 게이지 중앙 정렬 스타일
- 성장 단계 박스 스타일 (연한 민트 배경)
- 반응형 대응 (max-width: 720px, 모바일 패딩 조정)

## 레이아웃 구조

### 변경 전
```
[나의 정원]
[게이지] | [성장 단계 + 씨앗 이름]
[오늘의 정원 소식]
```

### 변경 후
```
[나의 정원]
[최근 30일 중 N일을 기록했어요.
 감정꽃이 자라고 있어요.]

        [게이지 중앙]
        
[현재 성장 단계]
[새싹 단계예요]

[씨앗 이름] [나의 씨앗] [✏️]

─────────────────
🌱 [오늘의 정원 소식: ...]
```

## 주요 변경 사항

### 1. 상단 텍스트 영역
- 좌측 정렬 유지
- 서브 텍스트를 2줄로 자연스럽게 줄바꿈 (`<br />` 추가)

### 2. 게이지 영역
- 중앙 정렬 (`justify-content: center`)
- 게이지 아래 여백 32px (모바일 24px)

### 3. 현재 성장 단계 섹션
- 연한 민트 배경 (`#f0faf7`)
- 라운드 박스 (`border-radius: 16px`)
- 중앙 정렬
- 작은 라벨 + 굵은 텍스트 구조

### 4. 씨앗 이름 섹션
- 중앙 정렬 (`justify-content: center`)
- 한 줄 배치
- 연필 아이콘 버튼 유지

### 5. 오늘의 정원 소식
- 하단 배치
- 좌측 정렬
- 아이콘 + 텍스트 구조
- 상단 divider 라인 추가
- 2줄 말줄임 처리 (`-webkit-line-clamp: 2`)

## 반응형 대응

### 모바일 (기본)
- 세로 스택 구조 그대로
- 패딩: 24px 16px
- 게이지 여백: 24px

### 태블릿/데스크톱
- 카드 `max-width: 720px`
- 중앙 정렬 (`margin: 0 auto`)
- 내부 구조 동일

## CSS 클래스 변경

### 추가된 클래스
- `.home-garden-header` - 상단 텍스트 영역
- `.home-garden-gauge-wrapper` - 게이지 래퍼 (중앙 정렬)
- `.home-garden-stage-box` - 성장 단계 박스
- `.home-garden-stage-label` - 성장 단계 라벨
- `.home-garden-stage-value` - 성장 단계 값
- `.home-garden-seed-section` - 씨앗 이름 섹션
- `.home-garden-seed-label` - 씨앗 이름 라벨
- `.home-garden-seed-value` - 씨앗 이름 값
- `.home-garden-message-wrapper` - 메시지 래퍼
- `.home-garden-message-icon` - 메시지 아이콘

### 제거된 클래스
- `.home-garden-content` (flex 좌우 배치)
- `.home-garden-left` (게이지 영역)
- `.home-garden-right` (텍스트 영역)
- `.home-garden-stage` (기존 단계 텍스트)
- `.home-garden-seed-row` (기존 씨앗 이름 줄)
